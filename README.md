3# EduWave UI/UX Scaffold

A mobile-first static website scaffold for an education platform with video, audio, and radio learning experiences.

## Included pages

- `index.html` — Landing page
- `signup.html` — Sign-up page
- `login.html` — Login page
- `dashboard.html` — Student dashboard
- `courses.html` — Course catalog
- `course-detail.html` — Course detail page
- `lesson-player.html` — Video lesson player page
- `audio-player.html` — Audio player page
- `radio.html` — Radio page
- `instructor-dashboard.html` — Instructor dashboard
- `profile.html` — Profile page

## Assets

- `styles.css` — Mobile-first responsive styling
- `script.js` — Minimal interactivity for tab switching and navigation

## How to preview

Open `index.html` in a browser to view the landing page. Use the built-in page links to navigate through the app.
